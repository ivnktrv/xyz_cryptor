**XYZ Encryptor** - программа, которая шифрует текст.

Команды:

```
           ─┐
xyz -h      |_  Справка
xyz --help  |
           ─┘                                            ─┐
xyz -e [файл.txt] [название файла без расширения]         |_  Зашифровать файл. Выходной файл будет иметь расширение .xyz
xyz --encrypt [файл.txt] [название файла без расширения]  |
                                                         ─┘
                         ─┐           
xyz -d [файл.xyz]         |_  Вывести расшифрованный текст на консоль
xyz --decrypt [файл.xyz]  |
                         ─┘
                                                     ─┐           
xyz -d [файл.xyz] --outfile [файл.расширение]         |_  Вывести расшифрованный текст на консоль
xyz --decrypt [файл.xyz] --outfile [файл.расширение]  |
                                                     ─┘
```
